# Initial setup guide

Download the [Raspberry Pi Imager](https://www.raspberrypi.com/software/)

Run base install GUI

Reboot

Launch ***Raspberry Pi Configuration***

* Edit host name
* Enable SSH
* Enable VNC

Reboot

Launch ***Terminal***
* `sudo apt update`
* `sudo apt upgrade`
* `sudo apt install fish emacs neofetch screenfetch` 
* `fish`
* `chsh -s (which fish)`
